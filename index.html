Project

Cage Game

This is the Cage Game I made by EasyX Graphics Library. In this game, I used 2D array to represent the location of each coordinate numerically in the map and I used a structure POS to represent the position of the character we control. Another important part is to create a control robot to detect the coordinate of next position by considering whether character is pushing a cage and whether the next position is destination or floor after user clicking the key.
